MY TECH STACK

HTML & CSS

I have a strong foundation in HTML and CSS. I use HTML to define the content and structure of my websites, while CSS enables me to style and customize the layout, colors, and typography.

JavaScript

I leverage JavaScript's power to enhance the interactivity and functionality of my websites. Whether it's implementing dynamic features, validating forms, or making AJAX requests, I utilize JavaScript to create responsive and engaging user experiences.

React JS

React's component-based architecture and virtual DOM enable me to create reusable and modular UI components. I leverage React's state management, routing, and lifecycle methods to build interactive and efficient web applications.

Laravel

Laravel is a popular open-source PHP framework that follows the Model-View-Controller (MVC) architectural pattern. It provides an expressive and intuitive syntax, making it enjoyable for developers to work with. Laravel offers a wide range of built-in features and tools for routing, caching, authentication, and database management, allowing for rapid application development.
